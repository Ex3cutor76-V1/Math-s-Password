# Math-s-Password
Literalmente um projeto que fiquei pensando em fazer após ver um vídeo de alguém fazendo um gerador de senha em python, admito, o código de fato não é meu eu só alterei algumas coisas e admito ter pedido ajuda algumas vezes para IA para pelo menos tentar entender os códigos em python e simplesmente aprender mais fácil, talvez o gerador de senhas tenha alterações no futuro assim que pelo menos eu aprender alguma coisa (Já que sou muito acostumado com HTML)

# Como o Math-s-Password funciona? 

No caso ele gera uma senha aleatória de 16 caracteres, usando letra minúscula, maiúscula números e caracteres especiais (Exemplo: #*&) misturando todos eles de forma aleatória com o módulo secrets garantindo que seja seguro.

# Instalação: Livre para todos os sistemas operacionais 

# No Linux:

Instale as dependências (Se caso você não tiver):

Debian, Ubuntu ou distros baseadas:
```bash
sudo apt install git
sudo apt install python3
```
Arch ou distros baseadas:
```bash
sudo pacman -S git 
sudo pacman -S python
```
Fedora ou Distros baseadas:
```bash
sudo dnf install git
sudo dnf install python3
```

Primeiro de tudo clone o repositório:
```bash
git clone https://github.com/Ex3cutor76-V1/Math-s-Password.git
```
Após isso use:

```bash
cd Math-s-Password/
```

Agora para usar use:

```bash
python3 Maths-Password.py 
```
